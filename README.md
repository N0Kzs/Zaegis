## About
Zaegis is a Crime Analytics Dashboard and Deployment Decision Support System. We harness the power of geographic data, crime pattern analysis, and predictive policing to optimize manpower allocation. We built this application from the ground-up to an actual implementation in a usable web app.

The system takes direct data feeds from official police stations and processes them to generate comprehensive crime analytics, actionable insights, and strategic deployment plans. The primary goal of the system is to assist law enforcement in making informed, data-driven decisions regarding public safety and resource allocation.
